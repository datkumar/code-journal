## First C program

```c title="code.c"
// Library imports
#include <stdio.h>

// This function is starting point of execution
void main () {
    printf("Hello World!\n");
}
```

```sh
# Compile "code.c" which generates an executable "a.out"
cc code.c
# Run the executable
./a.out

# Output:
Hello World!
```
