# Flutter App Assets

## App Icon Setup:

App Icon Generator: [https://www.appicon.co/](https://www.appicon.co/)

> Make sure the **icon file name matches asset in native code**

**For Android:**

Use `./android/app/src/main/res/` folder

If adding only **ONE** size, add it to the `.../res/drawable/` folder
